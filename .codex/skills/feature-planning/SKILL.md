# Feature Planning

Use when the user is refining a feature idea and needs pushback before implementation planning.

## Load These Sources

- `.claude/skills/feature-planning/SKILL.md`

## Operating Workflow

1. Use the canonical `.claude` skill as the main workflow.
2. Default to the user prompt and current thread before opening docs.
3. Read repo docs or code only when a decision depends on repo truth.
4. Challenge assumptions and narrow scope before drafting a plan.
5. Do not finish until the plan is decision-complete.
